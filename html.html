HTML'niň wezipesi

HTML (HyperText Markup Language) - web sahypalarynyň gurluşyny kesgitleýän standart bellik dilidir. Ol web brauzerlerine, sahypanyň nädip görkezilmeli we nähili gurluşy bolmalydygyny görkezyär.

Mysal üçin:

<!DOCTYPE html>
<html>
<head>
    <title>Sahypa ady</title>
</head>
<body>
    <h1>Salam TäsinDünýä!</h1>
    <p>Bu meniň ilkinji web sahypam!</p>
</body>
</html>
